Etiquetas de reemplazo

Utiliza las siguientes etiquedas para que sean reemplazadas con información de cada asistente al evento

  • **nombreCompleto** : Se reemplazará por el nombre completo del asistente.
  • **fechaActual** : Se reemplazará por la fecha de creación de esta constancia.
  • **evento** : Se reemplazará por el nombre del evento.
